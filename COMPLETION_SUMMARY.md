## 📋 Composants Livrés

### 1. DAGs Airflow (5 au total)

#### ✅ wikimedia_ingestion_kafka.py

- Classification: bots/humains, anonymes/connectés, types actions

#### ✅ wikimedia_traitement_agregations.py

- 5 fichiers JSON générés

  - bot_ratio.json

- Anomalies données (timestamps, langues invalides)
- Sortie: anomalies.jsonl (format JSONL)
- Fréquence: Toutes les heures
- Status: ✅ Testé et opérationnel

#### ✅ wikimedia_reporting.py
- Rapport consolidé quotidien

  2. Rapport qualité données
  3. Rapport trafic
  4. Rapport système
- Format: rapport_YYYY-MM-DD.json
- Fréquence: Quotidien à 2h du matin
- Status: ✅ Implémenté et documenté

#### ✅ wikimedia_monitoring.py (NOUVEAU)

- Fréquence: Toutes les 5 minutes

## 🚀 Commandes de Démarrage

### Docker (Production)
```bash
# Démarrer tous les services
docker-compose up -d

# Vérifier les services
docker ps

# Accéder à Airflow
# http://localhost:8080
# Identifiants: airflow/airflow
```

### Dashboard
```bash
# Générer données de test (optionnel)
python generate_test_data.py

# Démarrer serveur dashboard
python serve_dashboard.py

# Accéder au dashboard
# http://localhost:8000
```

### Déclencher DAGs (Docker)
```bash
# Déclencher ingestion
docker exec airflow-project-airflow-scheduler-1 airflow dags trigger wikimedia_ingestion_kafka

# Déclencher traitement
docker exec airflow-project-airflow-scheduler-1 airflow dags trigger wikimedia_traitement_agregations

# Déclencher détection
docker exec airflow-project-airflow-scheduler-1 airflow dags trigger wikimedia_detection_anomalies

# Déclencher reporting
docker exec airflow-project-airflow-scheduler-1 airflow dags trigger wikimedia_reporting

# Déclencher monitoring
docker exec airflow-project-airflow-scheduler-1 airflow dags trigger wikimedia_monitoring
```

### Lister les DAGs
```bash
docker exec airflow-project-airflow-scheduler-1 airflow dags list | grep wikimedia
```

## 🔧 Problèmes Résolus

### Issue 1: NameError - Missing dag decorator
**Solution**: Importation de `dag` depuis airflow.decorators dans tous les DAGs

### Issue 2: TypeError - schedule_interval deprecated
**Solution**: Remplacement par `schedule` parameter (Airflow 3.2.1+)

### Issue 3: ModuleNotFoundError - kafka missing
**Solution**: Dockerfile personnalisé avec pip install de requirements.txt

### Issue 4: Docker Volume - Data not persisting
**Solution**: Ajout volume mount `/tmp/airflow_wikimedia_data` dans docker-compose.yaml
