Demo
----

This folder contains screenshots used in the project demo. Add one or more images here and reference them from the main README. Keep filenames short, for example `screenshot-1.png`.

How to add a screenshot:

1. Save a dashboard screenshot as `docs/demo/screenshot-1.png`.
2. Commit the image and push the branch.

Example markdown reference to include in the main README:

![Dashboard demo](docs/demo/screenshot-1.png)

Do not include emojis or extra decorations in the README; keep the demo section concise.
